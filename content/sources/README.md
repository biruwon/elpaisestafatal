# Source records

Source records are reusable references to public documents. They are data records, not editorial bylines.

Required frontmatter:

```yaml
id: ine-ecv-2025
title: Encuesta de condiciones de vida 2025
url: https://example.org/source
date: 2026-02-05
type: official-report
```

Optional fields may describe access notes or the relevant section. Do not add publisher, author, editor or human-review fields.
