# Evidence records

Evidence records are reusable observations, definitions, calculations or legal references. A record can support multiple claims.

Required frontmatter:

```yaml
id: arope-2025-spain
kind: statistical-series
sourceIds: [ine-ecv-2025]
period: 2025
geography: España
unit: percent
```

The body should state the evidence, method and limitation. Regional records must identify the autonomous-community scope and comparability caveat.
